О сайте

Сервис прогноза погоды

Добро пожаловать на наш сервис прогноза погоды! Мы предоставляем точную и актуальную информацию о погоде для городов по всему миру.

Основные возможности:
  • Текущая погода и прогноз на 7 дней
  • Автодополнение при вводе названия города
  • История ваших поисков
  • Статистика популярных городов
  • REST API для разработчиков
Используемые технологии
Backend
  • Django (Python web framework)
  • PostgreSQL (база данных)
  • Python Decouple (конфигурация)
  • Requests (HTTP клиент)
Frontend
  • Bootstrap 5 (CSS framework)
  • Bootstrap Icons
  • Vanilla JavaScript
  • Django Templates
API
  • Open-Meteo API (данные о погоде)
  • Open-Meteo Geocoding API
Инфраструктура
  • Docker & Docker Compose
  • Poetry (управление зависимостями)
Источник данных
Open-Meteo

Бесплатный API прогноза погоды с открытым исходным кодом

Все данные о погоде предоставляются сервисом Open-Meteo — это надежный источник метеорологических данных, который агрегирует информацию от ведущих мировых метеослужб.

Особенности Open-Meteo:
  • Данные обновляются каждый час
  • Глобальное покрытие всех регионов мира
  • Высокая точность прогнозов
  • Бесплатное использование без API ключей
Подробнее о функциях
